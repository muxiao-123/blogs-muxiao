### Npm

```
npm node 升级

npm cache clean -f 清除缓存

npm install n -g // n模块 是管理node版本

n stable // 将node升级到最新稳定版

origin node v12.16.3 npm v6.2x
current node v16.16.0 npm v8.11.0

npm WARN config global `--global`, `--local` are deprecated. Use `--location=global` instead.

```
